# Image Compression using Low Rank Approximation

## Project Overview

In this GitHub project, we implement a simple image compressor using low rank approximation. We use numpy's SVD function, which automatically uses the optimized version of SVD (QR decomposition and Golub-Kahan Bidiagonalization). Here, we set k = 50 and focus on grayscale images.
